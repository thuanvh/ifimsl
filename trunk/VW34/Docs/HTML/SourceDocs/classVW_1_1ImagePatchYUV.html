Detailed Description

Author:
BJ Tordoff
Date:
28/03/2002 @ Brief A storage class for colour (YUV) image patches.

Functions are provided for creating patches from mono images using a supplied image position. If the position is supplied as floating point then bilinear interpolation is used to get the values. This doesn't cost much processing, and Smith:Sinclair:Cipolla:Wood:BMVC1998 indicate that it has some serious effect on the matching. However, I have some reservations about the fairness of their comparisons, so the choice is yours. Note that getting and setting of data is unchecked, so be careful of sizes.
