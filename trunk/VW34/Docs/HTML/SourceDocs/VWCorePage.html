VWCore

The Core Vision Library forms the heart of VW, providing image classes, image IO, feature extraction and geometric calculations.

Also, the core library contains abstract base classes for the sequencers and displays which are implemented in the more specific libraries. As it is the largest part of the library collection, the main user oriented bits are collected into modules as follows:

  • Image : Image classes, file IO, conversion from raw data and colour model conversions.
  • ImageProcessing : Operations on images (eg convolution, feature detection etc).
  • GeometricObjects : Objects to represent geometric constructs such as points, lines, rotations, multi-view relations etc.
  • GeometricAlgorithms : Algorithms and operations on geometric objects, such as calculating image-relations.
  • CoreSequencers : The base class and basic file-based sequencers. Sequencers provide stepped or continuous retrieval of sequential images.
  • CoreGraphics : The base classes for displays and basic (non-optimised) drawing functions. Also, functions for drawing things on images (including other images - ie. image compositing).
  • MathsStuff : Some mathematical utility functions and classes, such as random number generation, probability densities, Kalman filtering.
