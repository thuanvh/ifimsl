Detailed Description

a small class to hold the constants associated with a one-sided exponential distribution.

Author:
BJ Tordoff
Date:
03/03/2004 This PDF class allows evaluation of the probability density at any location, and sampling from a random variable with this PDF. Since some parts of the evaluation depend only on , storing as a class gives some computational benefits. The functions is:

prob(X==x) = 1/m * exp(-x/m)

where m is the free parameter. The mean and variance of the PDF are then

E[X] = m E[X^2] = 2m^2 V[X] = m^2

or,

mu = m sigma = m;
