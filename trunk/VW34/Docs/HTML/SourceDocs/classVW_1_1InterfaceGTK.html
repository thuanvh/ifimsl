Detailed Description

template<class WidgetType>
class VW::InterfaceGTK< WidgetType >

The abstract interface class.

All GTK interfaces derrive from this, and this class merely provides some of the house-keeping functions.

Author:
BJ Tordoff
Date:
02/08/2002 *ABSTRACT* base class for interfaces, providing common buttons and file selection stuff.
