Detailed Description

template<class Type>
class VW::ImageMono< Type >

A class for storing monochrome images.

Author:
ID Reid, BJ Tordoff
Date:
02/08/02 This image class stores the image in a row-major packed format (ie. like a standard C/C++ array), with pixels as single values of type Type.

Pointers to the start of each row.

A pointer to the start of the row r is given by image.pixel[r], and a pixel at row r, column c, by image.pixel[r][c]. This is functionally identical to saying image[r][c], but in some circumstances may be faster.
