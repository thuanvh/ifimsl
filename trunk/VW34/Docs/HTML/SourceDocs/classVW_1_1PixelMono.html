Detailed Description

template<class Type>
class VW::PixelMono< Type >

A class for storing Mono pixels.

Author:
ID Reid, BJ Tordoff
Date:
02/08/02 This pixel class stores luminance data only.

If not specified, the default template format is 8-bit, ie unsigned char, so that VW::PixelRGB<> p == VW::PixelRGB<unsigned char> p.
