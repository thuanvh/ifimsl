Compute a fundamental matrix from two lists of 2D point features.

Minimisation is performed using Levenberg-Marquadt with Sampson distance as the error measure. Note that no normalisation is performed, so its best if you normalise the data before calling this.

Returns:
true for successful minimisation, false otherwise (homography is left untouched in this case).
