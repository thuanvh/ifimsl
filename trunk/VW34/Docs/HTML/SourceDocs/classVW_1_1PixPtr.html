Detailed Description

template<class Type, int Skip>
class VW::PixPtr< Type, Skip >

A pixel pointer implementation.

Author:
ID Reid
Date:
02/08/02 The real purpose is to allow interleaved images to appear the same as ordinary ones by overloading the pointer arithmetic to skip over extra bytes.
