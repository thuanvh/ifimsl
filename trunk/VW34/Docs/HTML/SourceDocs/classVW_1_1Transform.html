Detailed Description

The base class from which all transforms are made.

Author:
JGH Knight
Date:
02/08/2002 The abstract class Transform is here merely to introduce some consistency between types of transforms and the sorts of information/methods they must all provide, rather than because they are strictly necessary. This is because we are using compile-time polymorphism (templates) for our compute functions, rather than run-time polymorphism (virtual functions). If you derive a new transform type from this base class you can be sure it will be possible to instantiate versions of the various compute classes from it.
