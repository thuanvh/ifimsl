Detailed Description

template<class T>
class VNL::NumericLimits< T >

Standard limits for numeric datatypes.

Implementation of the May 96 ANSI Draft Working Paper (DWP) numeric_limits class. Numbering in the documentation below refers to section 18.2 of the DWP.

Minimum positive denormalized value.

Meaningful for all floating point types. In specializations for which has_denorm == false, returns the minimum positive normalized value. For types with has_denorm == false, the member denorm_min() shall return the same value as the member min().

Minimum finite value.

Equivalent to CHAR_MIN, SHRT_MIN, FLT_MIN, DBL_MIN, etc.

For floating types with denormalization, returns the minimum positive normalized value, denorm_min().

Meaningful for all specializations in which is_bounded == true, or is_bounded == false && is_signed == false.

True if the type is modulo.
