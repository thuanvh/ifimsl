Detailed Description

Representation of a 3D quaternion.

Author:
BJ Tordoff
Date:
02/11/2002 This quaternion class represents a 3D rotation as a complex vector with three imaginary parts x,y,z and real part r: q = ix + jy + kz + r. Whilst designed for simlarity to the (partially incorrect) VNL quaternion class, it provides additional functions for rotating VW::Vector3D's and for initialisation from and recovery of Euler angles.

Note that unlike the VNL equivalent, it is guaranteed to be a unit vector.

As an IOObject descendant it can be streamed as binary, ASCII or XML-style.
