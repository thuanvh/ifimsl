Detailed Description

Simple timer class for measuring elapsed time, with precision down to microseconds.

Author:
BJ Tordoff
Date:
25/03/2004 There are several ways to use this class, but some general rules:
  • The timer starts when the class is created
  • Calling Start() will reset the timer to zero
  • Calling Stop() stores the end time
  • Calling GetAsXXX() with the timer running returns the elapsed time but leaves the timer running

Some example uses:

VW::Timer timer; t.Start(); SomeFunc(); t.Stop(); cout << "SomeFunc took " << timer << endl;

VW::Timer timer; SomeCode(); cout << "Time since start is " << timer << endl; SomeMoreCode(); cout << "Time since start is " << timer << endl;

VW::Timer timer; double secs1,secs2; timer.Start(); SomeCode(); secs1 = timer.GetAsSeconds(); timer.Start(); // Restart the timer SomeMoreCode(); secs2 = timer.GetAsSeconds();

cout << "Code took " << secs1 << " + " << secs2 << " seconds." << endl;
