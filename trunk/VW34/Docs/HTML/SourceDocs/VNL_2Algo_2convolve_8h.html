Detailed Description

Templated 1D and 2D convolution.

Author:
Peter Vanroose
Date:
22 August 2001

This file contains function declarations for 1D and 2D convolutions, both cyclic and non-cyclic, of VNL::Vectors and vnl_matrices. One can choose between straightforward `time-domain' implementations or using FFT to do a `frequency-domain' calculation.
