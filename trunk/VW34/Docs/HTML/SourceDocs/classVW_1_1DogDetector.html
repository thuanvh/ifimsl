Detailed Description

Interest point detection using David Lowe's extrema of Difference-of-Gaussian (DOG) operator.

Author:
ND Molton
Date:
15/10/2003 include <VW/Improc/dogdetector.h>

This is half of an implementation of David Lowe's SIFT (Scale invariant feature transform) feature detector/descriptor (the other half is GetSIFTDescriptor). The implementation is based on the following papers:
