Swap this image's data for other image data - size, ROI etc are also exchanged.

This is handy to avoid an image copy when doing things like:

void MyFunc(ImageXXX& im) { VW::ImageXXX result; DoSomeProcessing(im,result); im.ExchangeData(result); }

which would otherwise require a copy of the result data back to the original.

PixPtr< Type, Skip > * VW::ImageBase< Type , PixPtr< Type, Skip > >::pixel [inherited]

Pointers to the start of each row.

A pointer to the start of the row r is given by image.pixel[r], and a pixel at row r, column c, by image.pixel[r][c]. This is functionally identical to saying image[r][c], but in some circumstances may be faster.
