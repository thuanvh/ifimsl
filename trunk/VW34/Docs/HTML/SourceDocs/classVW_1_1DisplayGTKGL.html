Detailed Description

A display that makes use of a GTK window.

Author:
BJ Tordoff bjt
Date:
01.01.2001 This display opens a GTK window and attaches a DisplayGTKGLWidget to it. All drawing and stuff is forwarded to the enclosed widget.

Convenience function to draw a general ellipse.

The parameters are such that they are easily recovered from the matrix equation for an ellipse:

( x-x0 y-y0 ) C^-1 ( x-x0 ) = sigma^2 ( y-y0 )

where sigma gives the "radius" (number of std devs) and C is

C = [ rx 0 ] [ rx skew ] [ skew ry ] [ 0 ry ]

Parameters:
x0 x coord of centre
y0 y coord of centre
rx radius in x-direction at y=0
ry radius in y-direction at x=0
skew effect of y on x, where skew=0 gives a right ellipse
