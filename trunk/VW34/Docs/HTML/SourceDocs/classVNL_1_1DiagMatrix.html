Detailed Description

template<class T>
class VNL::DiagMatrix< T >

stores a diagonal matrix as a single vector.

DiagMatrix stores a diagonal matrix for time and space efficiency. Specifically, only the diagonal elements are stored, and some matrix operations (currently *, + and -) are overloaded to use more efficient algorithms.
