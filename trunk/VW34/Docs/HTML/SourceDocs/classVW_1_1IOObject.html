Detailed Description

Abstract base class for objects that can be input/output in a standard (streaming) way.

Author:
BJ Tordoff
Date:
22/10/02 This abstract class forces inheriting sub-classes to implement a set of streaming functions, providing input and output in binary, ASCII and XML-style ASCII formats. In addition it provides basic file versions of these functions that do not need to be overloaded, and maps the standard streaming operation to the ASCII functions.
