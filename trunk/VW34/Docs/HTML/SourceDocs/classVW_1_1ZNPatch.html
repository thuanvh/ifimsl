Use the (raw) values stored in the patch and calculate the zero-normalised values.

Optionally the values can be weighted using a Gaussian to emphasize the central region - this adds some computation time but can (supposedly) improve recall rates.

References std::sqrt(), and _pf_GaussianMask::weight.
