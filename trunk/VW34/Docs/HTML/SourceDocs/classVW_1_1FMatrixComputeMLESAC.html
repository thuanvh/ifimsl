Detailed Description

outlier rejection for fundamental matrices.

Author:
JGH Knight
Date:
02/11/02 As for Homography2Ds, this is a class derived from a version of ComputeMLESAC instantiated for FMatrix transforms. Also, the BaseFeature type is forced to be VW::Point2D, since fundamental matrices cannot be calculated from lines.

Derivation is used rather than just specialisation of the template so that additional features can be provided over and above that provided by ComputeMLESAC.
