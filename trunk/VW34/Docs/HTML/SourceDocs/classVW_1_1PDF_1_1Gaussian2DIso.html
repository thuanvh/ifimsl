Detailed Description

a small class to hold the constants associated with a 2D Isotropic Gaussian PDF (note that this is much faster than the non-isotropic VW::PDF::Gaussian2D).

Author:
BJ Tordoff
Date:
30/06/2003 This PDF class allows evaluation of the probability density at any location, and sampling from a random variable with this PDF. Since some parts of the evaluation depend only on mean and covariance, storing as a class gives some computational benefits.
