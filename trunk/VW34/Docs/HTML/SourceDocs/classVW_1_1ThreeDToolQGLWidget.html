const char *  name = "ThreeDToolQGLWidget" 
) [inline]

Construct a ThreeDToolQGLWidget in the standard QWidget manner.

This initialises the virtual camera to the default settings of ThreeDToolGLCommon, so you will most likely want to later call ThreeDToolGLCommon::SetCameraSize() and ThreeDToolGLCommon::SetCameraParameters() to set the correct virtual camera.

Parameters:
parent The parent widget. If set to 0 the widget will be displayed on its own in a new window; this is only useful if you have derived from this class to provide a drawing function
name The name of the widget (and the title if used as a window)
