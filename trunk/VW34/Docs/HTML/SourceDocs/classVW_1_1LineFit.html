Detailed Description

Fit lines to edge data.

Parameters:
edat the input edge data (as output from VW::Canny)
lines the resulting set of straight line segments
lambda_lim a threshold on line straightness based on ratio of minimum eigenval of scatter matrix to number of points in the line (default 0.5)
lengththresh the minimum line length to be included in the result
