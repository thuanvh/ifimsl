Detailed Description

EssentialMatrixCompute5Point.

Author:
G Vogiatzis, BJ Tordoff
Date:
19/04/2005 Five point minimal-set algorithm for computing essential matrices. Using only five correspondences results in a 4D nullspace and requires solution of a 10th degree polynomial (and hence produces between 0 and 10 essential matrices). Despite this, it's actually very fast!

Ref: Nister, D. ICCV2003 "An efficient solution to the five-point relative pose problem" (see also his thesis)
