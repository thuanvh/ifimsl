Detailed Description

A class to represent an affine fundamental matrix.

Author:
BJ Tordoff
Date:
08/04/2005 FMatrixAffine is a three-by-three degenerate matrix stored as the five non-zero elements Fa = [ 0 0 a ] [ 0 0 b ] [ c d e ]

It is automatically rank 2, so no extra constraints are required. The class includes functions for calculating epipolar lines from points, and for calculating epipoles.
