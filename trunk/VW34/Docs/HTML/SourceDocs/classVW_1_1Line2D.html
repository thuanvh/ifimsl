Detailed Description

Representation of a 2D (infinite) line.

Author:
ID Reid
Date:
02/08/02 Note that this definition of a line is infinite - that is, no end-points are stored. For a finite line, see VW::LineSeg2D. The data is stored as a 3D vector [a b c] defining the line ax + by + c = 0.

Computation routines are conatined in VW::Line2DCompute and some other operations in operations.h
