Detailed Description

A Wx-based interface to the firewire sequencer.

Author:
BJ Tordoff and AJ Davison
Date:
26/06/2004 This sequencer is a version of SequencerFirewire which also provides a WxWidgets interface. The sequencer functionality is identical to the parent class, but with added functions to control the interface. As it is just a Wx widget it can be embedded in any WxWidgets form in the usual manner.

void VW::SequencerBase::SetDelay ( int  d  )  [inline, virtual, inherited]

Set the delay between frames (only usually used for non-hardware sequencers).

This is the time (in milliseconds) for which the sequencer sleeps between frames: the total frame period will be this plus the time taken to acquire the new frame. By default the delay is set to 1ms (zero or less is not a valid setting).
