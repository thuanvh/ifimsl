Detailed Description

template<class Type>
class VW::PixelRGB< Type >

A class for storing RGB pixels.

Author:
ID Reid, BJ Tordoff
Date:
02/08/02 This pixel class stores the colour in packed BGRA format, with A specifying the alpha value (if used - which is not common). Note the ordering for little-endian CPU, which corresponds to the order used by the Matrox Meteor capture device and the X-windows display system. The extra alpha channel ensures that data is word-aligned in memory (ie. chunks of 4 bytes of multiples thereof).

If not specified, the default template format is 8-bit, ie unsigned char, so that VW::PixelRGB<> p == VW::PixelRGB<unsigned char> p.
