Detailed Description

a small class to hold the constants associated with a Rayleigh PDF.

Author:
BJ Tordoff
Date:
30/06/2003 This PDF class allows evaluation of the probability density at any location, and sampling from a random variable with this PDF. Since some parts of the evaluation depend only on , storing as a class gives some computational benefits. The functions is:

prob(X==x) = x/(s*s) * exp(-(x*x)/(2*s*s)

where s is the free parameter. The mean and variance of the PDF are then

mu = s * sqrt(pi/2) sigma^2 = 0.5 * s*s * (4-pi);
