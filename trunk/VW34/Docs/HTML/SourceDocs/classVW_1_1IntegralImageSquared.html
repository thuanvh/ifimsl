Detailed Description

template<class DataType>
class VW::IntegralImageSquared< DataType >

Implementation of integral image squared.

Author:
BJ Tordoff
Date:
03/04/2004 The class is templated on the internal storage type, which should be at least as big as the data-type of the image being integrated. Usually you should maintain signed-ness, such as: Image Type Integral Image Type "unsigned char" -> "unsigned int" or "long unsigned int" "char" -> "int" or "long int" "unsigned int" -> "long unsigned int" "int" -> "long int" "float" -> "float" or "double" "double" -> "double" or "long double"
