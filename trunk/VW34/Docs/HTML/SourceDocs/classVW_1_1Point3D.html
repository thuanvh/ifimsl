Detailed Description

Representation of a 3D position or vector.

Author:
BJ Tordoff
Date:
02/08/2002 Point2D represents a position in 3D Euclidean space, and provides the expected operations of addition, subtraction, multiplication by scalars etc. In addition it inherits from the IOObject and provides the standard streaming capabilities.
