Detailed Description

template<class maskType>
class VW::Convolve< maskType >

Author:
ID Reid, DW Murray
Date:
02/08/2002 @ Brief A class for performing convolutions on images.

If all you want to do is smooth an image, using VW::SmoothGaussian or VW::SmoothUniform may be easier (and faster).
