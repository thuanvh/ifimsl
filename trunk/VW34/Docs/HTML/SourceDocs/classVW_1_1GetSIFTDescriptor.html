Detailed Description

Interest point description using the method of Lowe.

Author:
BJ Tordoff after ND Molton
Date:
09/11/2004 include <VW/Improc/getsiftdescriptor.h>

Functions for adding a SIFT descriptor to a point set based on NDM's SIFT code.

Input is a list of location as output by VW::Plessey or VW::DOGDetector and the source image or image-pyramid. Output is a list of SIFTFeatures where each location generates one SIFTFeature per dominant orientation (typically either 1 or 2).

NB: Currently turning off orientation speeds things up a lot, but the weighting flag isn't used as yet.
