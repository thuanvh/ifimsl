Detailed Description

Describe image features by the surrounding patch (with normalisation).

Author:
BJ Tordoff
Date:
25/11/2004 include <VW/Improc/getpatchdescriptor.h>

The normalisation can optionally be weighted, giving a distribution which is still zero mean and unit variance, but where the central values are given more prominence (at the cost of some initialisation time).
