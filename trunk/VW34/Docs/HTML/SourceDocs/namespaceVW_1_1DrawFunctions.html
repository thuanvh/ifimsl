Detailed Description

Draw things on displays.

Author:
BJ Tordoff, JGH Knight
Date:
02/08/2002 This file can draw things on general displays using primitives only. The functions are therefore slow and an optimised version should be used where available (these are normally provided with the implementations of the display).
