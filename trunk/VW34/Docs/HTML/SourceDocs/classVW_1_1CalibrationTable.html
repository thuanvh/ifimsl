Detailed Description

A camera calibration look-up table.

Author:
BJ Tordoff
Date:
02/11/2002 This class can hold a single calibration instance for a camera, or a set of calibrations sampled at various zoom-lens motor positions. In the latter case the calibration parameters are used to fit polynomials so that interpolated values can be retrieved.

CameraCalibration inherits from IOObject and is able to read calibration data from suitably formatted data files. The data is input/output in the following order:

  • Number of calibration data points
  • Number of distortion data points
  • Array of calibration data
    • motor position
    • focal-length
    • aspect
    • skew
    • principal-x
    • principal-y
  • Array of distortion data
    • motor position
    • kappa
