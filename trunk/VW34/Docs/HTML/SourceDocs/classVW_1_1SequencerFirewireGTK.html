Detailed Description

A GTK1-based interface to the firewire sequencer.

Author:
ID Reid, BJ Tordoff
Date:
02/08/02

(c) Copyright University of Oxford All rights reserved

This sequencer is a version of SequencerFirewire which also provides a GTK interface. The sequencer functionality is identical to the parent class, but with added functions to control the interface. As it is just a GTK1 widget it can be embedded in any GTK1 form, either as a button and pop-up window or as a full widget.

void VW::SequencerBase::SetDelay ( int  d  )  [inline, virtual, inherited]

Set the delay between frames (only usually used for non-hardware sequencers).

This is the time (in milliseconds) for which the sequencer sleeps between frames: the total frame period will be this plus the time taken to acquire the new frame. By default the delay is set to 1ms (zero or less is not a valid setting).
