Basic constructor.

Compute_obj name determines the name of the lowest-level block to find in the file. Once found, parameters can be added to be either written directly into the block (AddParameter), or added between a ...

pair (AddChildParam).

All-in-one constructor.

Compute_obj name determines the name of the lowest-level block to find in the file. Once found, the parameters listed in parameter[] are added as children (ie. between ...

pairs). In addition, the file is written to disk, allowing one-line parameter writing.
