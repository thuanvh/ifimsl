Detailed Description

Interest point detection using David Lowe's SIFT (Scale invariant feature transform) descriptor.

Author:
ND Molton
Date:
15/10/2003 This is an implementation of David Lowe's SIFT (Scale invariant feature transform) feature detector and a simple feature matcher. The implementation is based on the following papers:

{ lowe99object, author = "David G. Lowe", title = "Object Recognition from Local Scale-Invariant Features", booktitle = "Proc. of the International Conference on Computer Vision {ICCV}, Corfu", pages = "1150-1157", year = "1999", url = "citeseer.nj.nec.com/lowe99object.html" }

{ keypoints-distinctive, author = "David G. Lowe", title = "Distinctive image features from scale-invariant keypoints", note = "submitted to International Journal of Computer Vision, June 2003", url = "citeseer.nj.nec.com/582422.html" }

Classes/Structures/Functions are:

SiftFeature - The feature vector SiftParameters - Parameters of the SIFT Detector Sift - The detector SiftMatchNN - A basic nearest neighbour feature matching function
