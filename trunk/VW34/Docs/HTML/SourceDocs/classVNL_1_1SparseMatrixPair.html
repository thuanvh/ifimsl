Detailed Description

template<class T>
class VNL::SparseMatrixPair< T >

Stores elements of sparse matrix.

Only those values which are non-zero are stored. The sparse matrix currently supports only getting/putting elements, and multiply by vector or another sparse matrix.

Each row is stored as a vector of vcl_pair<unsigned int,T>, where the first of the pair indicates the column index, and the second the value. All rows are stored, as std::vector< row >;
