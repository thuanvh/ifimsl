Detailed Description

an abstract class that defines functions that all 1D PDF's have.

Author:
BJ Tordoff
Date:
30/06/2003 Note that two PDF's with different means or variances are different PDF's, and so no methods are provided for changing the mean and variance post-construction. This is deliberate, as it would remove much of the utility of the classes for fast evaluation/sampling.
