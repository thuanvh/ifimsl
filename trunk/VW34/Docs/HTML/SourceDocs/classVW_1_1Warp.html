Detailed Description

Warp a 2D image according to homography H.

Author:
ID Reid
Date:
02/08/2002 include <VW/Improc/warp.h>

Given an invertible 2D projective transformation (ie. non-singular 3x3 matrix) H, every point in the target image can be calculated from the input image.

Note that H must NOT be normalised - it must be in image coordinates, as must the distortion (if any). If you only want to correct distortion, use the DistortImage class.

Usage: VW::Warp w(H); w.Compute(input_image, output_image); or VW::Warp(input_image, H, output_image);
