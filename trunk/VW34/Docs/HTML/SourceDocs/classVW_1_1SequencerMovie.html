Detailed Description

A sequencer for handling a few types of Movie file.

Author:
ID Reid, BJ Tordoff
Date:
02/08/02 As this uses a hacked version of xanim, you'll need to get a copy. This sequencer can read images from any movie that xanim can handle.

void VW::SequencerBase::SetDelay ( int  d  )  [inline, virtual, inherited]

Set the delay between frames (only usually used for non-hardware sequencers).

This is the time (in milliseconds) for which the sequencer sleeps between frames: the total frame period will be this plus the time taken to acquire the new frame. By default the delay is set to 1ms (zero or less is not a valid setting).
