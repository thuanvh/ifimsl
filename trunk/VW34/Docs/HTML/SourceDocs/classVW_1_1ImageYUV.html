Detailed Description

template<class Type>
class VW::ImageYUV< Type >

A class for storing YUV (YCrCb?) images.

Author:
ID Reid, BJ Tordoff
Date:
02/08/02 This image class stores the image in a row-major format (ie. like a standard C/C++ array), with pixels as YUVA where A is a spare (alpha?) channel. This way pixels stored using 8-bit integers (ie. unsigned chars, as are produced by many cameras, and used on most displays) are aligned on 4-byte memory locations, adding a tad to performance. Strictly speaking this (and the related conversions) are really Y Cr Cb rather than YUV, but looking around the 'net the terminology seems to be interchangeable.

A pointer to the start of the row r is given by image.pixel[r], and a pixel at row r, column c, by image.pixel[r][c]. This is functionally identical to saying image[r][c], but in some circumstances may be faster.
