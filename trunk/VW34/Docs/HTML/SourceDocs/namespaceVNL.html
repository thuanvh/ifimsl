Convolve two VNL::Vector<T>'s, possibly with different base types T.

.

The returned VNL::Vector has base type U (the third argument). All calculations are done with type U, so take care! To specify the third argument, pass e.g. a null pointer, casted to U*. Thus: vnl_convolve(v1, v2, (double*)0) would convolve v1 and v2, and return a VNL::Vector<double>.
