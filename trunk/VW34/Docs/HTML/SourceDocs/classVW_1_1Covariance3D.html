VW::Covariance2D VW::Covariance3D::LineariseAroundHomgPoint ( const double  x,
const double  y,
const double  w 
) const

Treat this 3D covariance as a 2D homogeneous one and linearise it around the point (x,y,w) to get the 2D non-homogeneous one.

If the point is linearised as [u,v] = [x/w, y/w] then the covariance linearises as C_2 = J C J' where J = [ w 0 -x ] / w^2 [ 0 w -y ] This is optimised to take 26 flops (standard multiply would take 45).
