Detailed Description

Class to hold the bounds on the Region of Interest (ROI) of a 2D array (image).

Author:
ID Reid, BJ Tordoff
Date:
15/01/2003

(c) Copyright University of Oxford All rights reserved

Data is stored as the first row/column of interest, and one beyond the last, so a loop that does for(x=roi.Left();x<roi.Right();x++) {} does the right thing.
