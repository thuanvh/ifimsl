Detailed Description

Interest point detection using the method of Harris/Plessey.

Author:
ID Reid and BJ Tordoff
Date:
02/08/2002 The Plessey Corner Detector (with Harris-Stephens scoring system)

the Plessey (or Harris) corner detection system. The main detector is a function object 'Plessey' which takes an image or image-pyramid as input and produces a list of interest point locations (ie. vector<CornerFeature>).

To convert these locations into useful features, this list must then be passed to a descriptor finder, eg:

vector<CornerFeature> corners; vector<SIFTFeature> siftcorners;

VW::PlesseyDetector(image, corners); VW::GetSIFTDescriptor(corners, image, siftcorners);

Although this seems wasteful, the memory overhead in calculating two lists is not significant, and some descriptors (such as SIFT) can produce more than one feature per location.
