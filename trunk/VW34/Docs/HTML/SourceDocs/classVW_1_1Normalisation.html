Detailed Description

A 2D normalisation object.

Author:
JGH Knight
Date:
02/11/2002 This class is designed for normalising points and lines on the 2D plane, such as must be done before calculating the various image transforms. A variety of normalisations are possible, but the most common and simplest is to simply provide the image width (assuming a 4x3 ratio image), giving a normalisation to coords (-1<x<1, -0.75<y<0.75).

X and Y are scaled equally. Doing otherwise complicates things and is rarely needed.
