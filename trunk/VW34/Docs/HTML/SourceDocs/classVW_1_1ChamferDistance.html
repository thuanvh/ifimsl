Detailed Description

Class for computing a linear distance transfor (aka chamfer distance) image.

Author:
BJ Tordoff
Date:
26/06/2003 The distance transform uses a binary image of "features" from which the distance is measured. There are several ways in which it can be used, the easiest of which is to provide a binary image with features having a "true" value and everything else "false". Typically it is used on a thresholded silhouette or on the output of the Canny edge detector (see VW::Canny), and compute functions for these types of data are also provided.
