%VW : Source-code documentation

0.34

The VW project aims to provide a lightweight set of classes and fast functions for performing basic computer-vision and image-processing related tasks. This documentation has been built from the source-code of the library and as such is quite technical in nature - for a more gentle introduction to all things VW see the user guide. Below is an index into the library by functionality (section Functionality), but first a list of the individual libraries that make up the Workbench. (A search utility is available here.)

Libraries

VW is split into several component libraries of increasing dependance:

Core libraries

  • VNL : A reduced version of the VXL project's numerics library
  • VW : The core functionality (really should be called VWCore). Image classes, image IO etc.

User interface libraries

  • VWX : 2D user interface functionality using raw X-windows commands.
  • VWXForms : 2D user interface functionality using the XForms widget library.
  • VWGL : 3D user interface functionality using OpenGL
  • VWGTK : 2D and 3D user interface functionality using GTK--.
  • VWGLOW : 2D and 3D user interface functionality using GLOW.
  • VWQt : 3D user interface functionality using Qt.

Hardware support

  • VWFirewire : Support for image capture from IEEE1394 digital cameras
  • VWMeteor : Support for image capture from the Matrox Meteor analogue capture card
  • VWYorick : Control of the Yorick series of active vision platforms
  • VWGTI : Control of the GTI mobile robot

Functionality

Perhaps more usefully, the Workbench is indexed by category below. Classes that provide related functionality are grouped by their function rather than by which library they can be found in (although the two are often related).

  • Sequencers : Classes for retrieving images from a range of sources, including file lists, movies, IEEE1394 cameras and the Matrix Meteor.
  • Image : Image classes, IO and conversions.
  • ImageProcessing : Image filtering, mangling and feature detection.
  • GeometricObjects : Objects to represent geometric constructs such as points, lines, rotations, multi-view relations etc.
  • GeometricAlgorithms : Algorithms and operations on geometric objects, such as calculating image-relations.
  • Displays : Classes for opening windows and drawing 2D things using a range of widget libraries.
  • GLTools : Classes that provide 3D OpenGL displays and 3D objects to draw on them.
  • UserInterface : Classes that provide widgets for controlling/viewing various VW objects.
