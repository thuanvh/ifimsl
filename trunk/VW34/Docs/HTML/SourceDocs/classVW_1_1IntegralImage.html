Detailed Description

template<class DataType>
class VW::IntegralImage< DataType >

Implementation of integral image as described in Viola's CVPR2001 paper on face detection via boosting.

Author:
ID Reid
Date:
06/08/2002 The class is templated on the internal storage type, which should be at least as big as the data-type of the image being integrated. Usually you should maintain signed-ness, such as: Image Type Integral Image Type "unsigned char" -> "unsigned int" or "long unsigned int" "char" -> "int" or "long int" "unsigned int" -> "long unsigned int" "int" -> "long int" "float" -> "float" or "double" "double" -> "double" or "long double"
